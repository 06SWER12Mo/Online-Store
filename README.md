# 🚀 Online Store - Spring Boot Backend

A complete e-commerce backend built with **Spring Boot 3.4.2** featuring JWT authentication, role-based access control, inventory management, order processing, payment integration, shipping management, and comprehensive analytics.

## ✨ Features

- 🔐 JWT Authentication with Refresh Tokens
- 👥 Role-Based Access Control (Admin, Manager, User)
- 🛍️ Product Management with Variants & Specifications
- 🛍️ Category & Subcategory Management.
- 📦 Order Management with Status Tracking
- 💳 Payment Processing & Refunds
- 🚚 Automated Shipping with Batch Processing on Big Areas
- 📊 Analytics Dashboard
- ⭐ Product Reviews & Ratings
- 📋 Receipt & Supplier Management
- 🖼️ Image Upload for Products, Categories & Users
- 📈 Real-time Inventory Management
- 🗺️ Location & Delivery Address Management

### Business Flow
Cart → Checkout → Payment → Order → Shipping → Delivery
